      Klepsidra.Cldr.List (klepsidra v0.1.0)

      Cldr backend module that formats lists.

      If we have a list of days like ["Monday", "Tuesday", "Wednesday"] then we can format that list for a given locale by:

      iex> Klepsidra.Cldr.List.to_string(["Monday", "Tuesday", "Wednesday"], locale: "en")
{:ok, "Monday, Tuesday, and Wednesday"}

      intersperse(list, options \\ [])

      @spec intersperse([term()], Keyword.t()) ::
  {:ok, list()} | {:error, {module(), String.t()}}

      Intersperces a list elements into a list format according to the list pattern rules for a locale.

      This function can be helpful when creating a list from Phoenix safe strings which are of the format {:safe, "some string"}

      Arguments

      • list is any list of of terms that can be passed through Kernel.to_string/1

      • options is a keyword list

      Options

      • :locale is any configured locale. See . The default is Klepsidra.Cldr.known_locale_names/0.

      • :format is one of those returned by Cldr.List.known_list_formats/0. The default is format: :standard

      Examples 

      iex> Klepsidra.Cldr.List.intersperse(["a", "b", "c"], locale: "en")
{:ok, ["a", ", ", "b", ", and ", "c"]}

iex> Klepsidra.Cldr.List.intersperse(["a", "b", "c"], locale: "en", format: :unit_narrow)
{:ok, ["a", " ", "b", " ", "c"]}

iex> Klepsidra.Cldr.List.intersperse(["a", "b", "c"], locale: "fr")
{:ok, ["a", ", ", "b", " et ", "c"]}

iex> Klepsidra.Cldr.List.intersperse([1,2,3,4,5,6])
{:ok, [1, ", ", 2, ", ", 3, ", ", 4, ", ", 5, ", and ", 6]}

iex> Klepsidra.Cldr.List.intersperse(["a"])
{:ok, ["a"]}

iex> Klepsidra.Cldr.List.intersperse([1,2])
{:ok, [1, " and ", 2]}

      to_string(list, options \\ [])

      @spec to_string([term()], Keyword.t()) ::
  {:ok, String.t()} | {:error, {module(), String.t()}}

      Formats a list into a string according to the list pattern rules for a locale.

      Arguments

      • list is any list of of terms that can be passed through Kernel.to_string/1

      • options is a keyword list

      Options

      • :locale is any configured locale. See . The default is Klepsidra.Cldr.known_locale_names/0.

      • :format is one of those returned by Cldr.List.known_list_formats/0. The default is format: :standard

      Examples 

      iex> Klepsidra.Cldr.List.to_string(["a", "b", "c"], locale: "en")
{:ok, "a, b, and c"}

iex> Klepsidra.Cldr.List.to_string(["a", "b", "c"], locale: "en", format: :unit_narrow)
{:ok, "a b c"}

iex> Klepsidra.Cldr.List.to_string(["a", "b", "c"], locale: "fr")
{:ok, "a, b et c"}

iex> Klepsidra.Cldr.List.to_string([1,2,3,4,5,6])
{:ok, "1, 2, 3, 4, 5, and 6"}

iex> Klepsidra.Cldr.List.to_string(["a"])
{:ok, "a"}

iex> Klepsidra.Cldr.List.to_string([1,2])
